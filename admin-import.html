Import existing customers

Paste your customer rows below, preview, then import into the database. Run this once for your existing customers so their renewals are detected.

The same value you set as ADMIN_KEY in Render.

Format per line: username password expire email (same order as your panel)
Separators: single space, tab, comma, or semicolon — all work. Expire is YYYY-MM-DD (optional).
Rows without an email are skipped (they can't be matched on renewal).
A header line containing username is detected and ignored.
